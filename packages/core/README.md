# @toryo/core

Core engine for the Toryo intelligent agent orchestrator.

Provides the orchestration loop, task management, quality ratcheting, trust-based delegation, and smart truncation that power Toryo.

## Installation

```bash
npm install @toryo/core
```

## Usage

```ts
import { createOrchestrator } from '@toryo/core';
```

See the [main Toryo README](https://github.com/JesseRWeigel/toryo#readme) for full documentation.
